스파이 발견!

$n(n \ge 3)$ 명 중에서 스파이를 찾으세요.

첫 번째 줄에는 테스트 케이스의 개수 $t(1 \le t \le 100)$가 주어집니다. 각 테스트 케이스는 다음과 같이 구성됩니다.

각 테스트 케이스의 첫 번째 줄에는 정수 $n(3 \le n \le 100)$이 주어집니다.

각 테스트 케이스의 두 번째 줄에는 $n$ 개의 정수 $a_1, a_2, \cdots, a_n(1 \le a_i \le 100)$이 주어집니다.

각각의 테스트 케이스마다, 스파이의 번호를 출력하세요.

첫 번째 테스트 케이스에서 스파이는 $13$이고 $13$은 $2$번째에 위치합니다.
